Manual do Usuário

Controles da Luta: Use os botões na parte inferior para iniciar/pausar o cronômetro, ir para o próximo round, reiniciar ou finalizar a luta.

Registro de Ações: Selecione o atleta ativo usando o seletor (Vermelho/Azul). Toque nos botões de ação (chute, soco, etc.) para abrir o menu de detalhes e registrar uma ação.

Editar/Remover Ação: Pressione e segure uma ação na lista para abrir o menu de edição. Lá você pode alterar os detalhes, o tempo da ação ou removê-la.

Reordenar Ações: Arraste e solte uma ação na lista para reordená-la. O tempo da ação será ajustado automaticamente para se encaixar na nova posição.

Tela Cheia: Use o botão de tela cheia para uma experiência imersiva. Em dispositivos iOS, você precisará primeiro "Adicionar à Tela de Início" pelo menu de compartilhamento do Safari.

Finalizar Luta: Ao finalizar, você terá opções para gerar um relatório em PDF, um arquivo JSON com os dados brutos, adicionar comentários e compartilhar a análise com uma IA para obter estratégias.
